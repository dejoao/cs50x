# cs50x

## O CS50x

O CS50x é uma versão online do famoso curso de introdução à ciência da computação de Harvard. Ele ensina os conceitos fundamentais da programação e da ciência da computação, cobrindo temas como algoritmos, estruturas de dados, desenvolvimento web, segurança, e muito mais. Ao longo do curso, os alunos aprendem várias linguagens de programação, incluindo C, Python, SQL, HTML, CSS e JavaScript, por meio de projetos práticos.
